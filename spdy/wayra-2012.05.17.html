¿Qué es SPDY?

Leandro López

HTTP está viejo

  • No se actualiza hace más de una década.
  • Es sincrónico.
  • No mantiene conexiones.
  • Demasiado trabajo para reducir la cantidad de requests.

SPDY

www.modspdy.com

  • Inventado y utilizado por Google
  • Usar cualquier servicio de Google con Chrome significa usar SPDY
  • Lo convierte en uno de los protocolos más probados de la historia
  • Conexión permanente
  • Seguro por definición
  • Server Push
  • Fácil de instalar en Apache
  • Próximamente nginx

Seguro por definición

  • SSL + NPN = HTTPS o SPDY
  • Ventaja adicional: prevenir session hijacking
  • Penalidad de velocidad solamente en la primer y única conexión

Veloz

  • Una sola conexión
  • Asincrónica
  • En ambas direcciones
  • Compresión de contenidos y cabeceras con un diccionario estático
  • chrome://net-internals/#spdy

Server Push

  • Al enviar una hoja de estilos, hacer push de todas las imágenes que haga referencia.
  • Enviar desde el servidor contenido directamente a la cache del navegador antes de que este la solicite.
